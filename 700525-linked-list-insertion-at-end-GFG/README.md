# [Linked List Insertion At End](https://www.geeksforgeeks.org/problems/linked-list-insertion-1587115620/1)
## Easy
You are given the head of a Singly Linked List and a value x, insert that value x at the end of the LinkedList and return the head of the modified Linked List.
Examples :
Input: x = 6, &nbsp; Output: 1 -&gt; 2 -&gt; 3 -&gt; 4 -&gt; 5 -&gt; 6
Explanation: We can see that 6 is inserted at the end of the linkedlist. &nbsp; 

Input: x = 1,   Output: 4 -&gt; 5 -&gt; 1Explanation: We can see that 1 is inserted at the end of the linked list.      
Constraints:0 ≤ number of nodes ≤ 1050 ≤ node-&gt;data , x ≤ 103
